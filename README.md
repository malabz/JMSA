### Usage

```shell
usage: java -jar  [-m] mode [-i] path [-o] path

  necessary arguments: 
    -i  Input file path (nucleotide sequences in fasta format)
    -m  two option (1.TreeAlign 2.CenterAlign)

  optional arguments: 
    -o  Output file path (default: inputfile.aligned)
```
